Quick Installation

One-Line Install

curl -fsSL https://raw.githubusercontent.com/CentaPoint/dabira-agent-install/main/install.sh | bash

This script will automatically detect your platform, download the correct binary, verify checksums, and install to /usr/local/bin.

Install Specific Version

curl -fsSL https://raw.githubusercontent.com/CentaPoint/dabira-agent-install/main/install.sh | bash -s -- --version v1.2.0

PowerShell Install

Invoke-WebRequest -Uri "https://github.com/CentaPoint/dabira-server-agent/releases/latest/download/dabira-agent-windows-x86_64.exe" -OutFile "dabira-agent.exe" Move-Item dabira-agent.exe C:\Windows\System32\

Download Binaries

Linux x86_64 Linux ARM64 macOS Intel macOS Apple Silicon Windows x64

Manual Installation Steps

# Download (Linux x86_64 example) curl -LO https://github.com/CentaPoint/dabira-server-agent/releases/latest/download/dabira-agent-linux-x86_64 # Make executable chmod +x dabira-agent-linux-x86_64 # Install sudo mv dabira-agent-linux-x86_64 /usr/local/bin/dabira-agent # Verify dabira-agent --version

Verify Installation

curl -fsSL https://raw.githubusercontent.com/CentaPoint/dabira-agent-install/main/verify.sh | bash

Features

  • Interactive setup wizard for easy configuration
  • Non-interactive setup for CI/CD and automation
  • Support for PostgreSQL, MySQL, and MongoDB
  • Secure credential storage using platform keyring
  • Beautiful terminal output with colors and tables
  • JSON output mode for scripting and automation
  • Daemon mode for long-running server deployments
  • Comprehensive health checks and diagnostics
  • Cross-platform: Linux, macOS, Windows

System Requirements

Operating System: Linux (glibc 2.27+), macOS 10.15+, Windows 10+
Architecture: x86_64 or ARM64
Memory: 128 MB minimum
Disk Space: 50 MB for binary
