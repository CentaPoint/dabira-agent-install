Dabira Agent

Secure database proxy for AI-powered analytics. Connect your databases safely to Dabira Cloud.

Quick Installation

Copy and paste this command in your terminal:

Linux / macOS
curl -fsSL https://raw.githubusercontent.com/CentaPoint/dabira-agent-install/main/install.sh | bash
What this does: Automatically detects your platform, downloads the correct binary, verifies checksums, and installs to /usr/local/bin

curl -fsSL https://raw.githubusercontent.com/CentaPoint/dabira-agent-install/main/install.sh | bash -s -- --version v1.2.0

Installation Steps

Linux / macOS
# Download (example: Linux x64) curl -LO https://github.com/CentaPoint/dabira-server-agent/releases/latest/download/dabira-agent-linux-x86_64 # Make executable chmod +x dabira-agent-linux-x86_64 # Install sudo mv dabira-agent-linux-x86_64 /usr/local/bin/dabira-agent # Verify dabira-agent --version

Run Dabira Agent in a Docker container:

Docker Run
docker run -d \ --name dabira-agent \ -v ~/.config/dabira-agent:/root/.config/dabira-agent \ -e DABIRA_REGISTRATION_TOKEN=your_token_here \ ghcr.io/centapoint/dabira-agent:latest
Note: Docker image coming soon. Subscribe to releases for updates.

PowerShell installation for Windows:

PowerShell
Invoke-WebRequest -Uri "https://github.com/CentaPoint/dabira-server-agent/releases/latest/download/dabira-agent-windows-x86_64.exe" -OutFile "dabira-agent.exe" Move-Item dabira-agent.exe C:\Windows\System32\
Windows Defender: You may see a SmartScreen warning. Click "More info" then "Run anyway". The binary is not yet code-signed.

Getting Started

After installation, configure your agent:

Setup
# Interactive setup wizard dabira-agent setup # Check status dabira-agent status # View all commands dabira-agent --help

System Requirements

Operating System Linux (glibc 2.27+), macOS 10.15+, Windows 10+
Architecture x86_64 (64-bit) or ARM64
Memory 128 MB minimum, 256 MB recommended
Disk Space 50 MB for binary
Network Outbound HTTPS (443) to api.dabira.com

Verify Installation

Verify the integrity of your installation:

Verification
curl -fsSL https://raw.githubusercontent.com/CentaPoint/dabira-agent-install/main/verify.sh | bash

This checks:

  • Binary checksum matches official release
  • Version is up to date
  • Basic commands work correctly
